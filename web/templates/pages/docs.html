Authentication

The API is open — no key required for basic usage. For higher rate limits, sign up for a free API key.

Send your key via any of these methods:

Authorization: Bearer shrike_your_key_here

X-API-Key: shrike_your_key_here

?api_key=shrike_your_key_here

Rate Limits

Shrike is a young service. Limits are strict while we scale.

Automated tools (curl, python-requests, etc.) require an API key.

Async Lookups

When you query a domain Shrike hasn't seen before, it crawls WHOIS servers live. This takes 3-10 seconds.

GET /api/v1/domains/new-domain.com

→ If known: returns data immediately (200)

→ If unknown: returns 202 with poll URL

{
  "status": "fetching",
  "poll_url": "/api/v1/domains/new-domain.com/status",
  "estimated_seconds": 5
}

Poll the URL every 2s until status is "ready", then fetch the full result.

Or add ?wait=true to block until data is available (up to 15s).

Domain WHOIS

GET /api/v1/domains/:name

Current WHOIS snapshot for a domain. Triggers live crawl if unknown.

GET /api/v1/domains/:name/history

All WHOIS snapshots over time. Params: from, to, limit, offset

GET /api/v1/domains/:name/status

Poll endpoint for async lookups. Returns fetching/ready/error.

GET /api/v1/domains/:name/dns

Current DNS records. Param: type (A, AAAA, MX, NS, TXT, CNAME)

IP WHOIS

GET /api/v1/ips/:address

Current WHOIS for an IP. Auto-finds the containing CIDR block.

GET /api/v1/ips/:address/domains

Reverse DNS history — domains that have pointed to this IP.

ASN Records

GET /api/v1/asns/:number

Current ASN info and announced prefixes.

GET /api/v1/asns/:number/prefixes

Currently announced IP prefixes for this ASN.

Search

GET /api/v1/search

Unified search across all pillars. Params: q, type (domain, ip, asn)

No PII search — you cannot search by personal email, phone, or address.

API Key Signup

POST /api/v1/keys/signup

Create a free API key. No payment required.

curl -X POST https://shrike.mp.ls/api/v1/keys/signup \
  -H "Content-Type: application/json" \
  -d '{"owner": "Your Name", "email": "you@example.com"}'

Returns a key starting with shrike_. Store it securely — it cannot be retrieved again.
